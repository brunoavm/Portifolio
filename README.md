# 🚀 Portfolio — Bruno Magalhães

> Portfólio profissional moderno desenvolvido com **HTML5, CSS3 e JavaScript puro**.

## ✨ Funcionalidades

- 🌙 **Dark / Light Mode** com preferência salva no localStorage
- 🎇 **Canvas de partículas** interativo no hero
- ⌨️ **Efeito typing** no subtítulo
- 🖱️ **Parallax** suave ao mover o mouse
- 📜 **Animações ao scroll** via Intersection Observer
- 📱 **Menu mobile** com drawer animado
- 🔍 **Filtro de projetos** por categoria
- ✅ **Formulário de contato** com validação
- 🔝 **Botão voltar ao topo**
- ♿ **Acessibilidade** básica (aria-labels, semântica HTML5)
- ⚡ **Performance** otimizada — zero frameworks, zero build

<br />

## 🛠️ Tecnologias

- **HTML5** — Estrutura semântica e acessível
- **CSS3** — Variáveis, Grid, Flexbox, animações e responsividade
- **JavaScript ES6+** — Vanilla JS, Canvas API, Intersection Observer, requestAnimationFrame
- **Remix Icon** — Biblioteca de ícones via CDN
- **Google Fonts** — Oxanium e Plus Jakarta Sans

<br />

## ⚙️ Como rodar localmente

Não precisa de instalação ou build. Basta:

```bash
# Clone o repositório
git clone https://github.com/brunoavm/portifolio.git

# Abra a pasta no VS Code
code portfolio/

# Abra o index.html com a extensão Live Server
# ou abra diretamente no navegador
```

## 🎨 Seções do Site

| Seção | Descrição |
|---|---|
| **Hero** | Apresentação com partículas, typing effect e avatar animado |
| **Sobre** | Quem sou, trajetória e stack |
| **Skills** | Cards de tecnologias com barras de nível animadas |
| **Projetos** | Projetos em destaque, pessoais e simples com filtros |
| **Jornada** | Timeline da trajetória de aprendizado |
| **Contato** | Links, redes sociais e formulário |
